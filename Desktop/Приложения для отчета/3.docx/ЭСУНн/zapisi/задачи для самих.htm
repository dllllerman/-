7.    Задачи для самостоятельного решения

Массивы записей

Вариант 1

a)       Сформировать массив записей, содержащий информацию о поездах, отправляющихся с Ленинградского вокзала г. Москвы (номер поезда, станция назначения, время отправления, время в пути).

b)       Вывести на экран содержимое массива в виде таблицы с заголовками столбцов.

c)        Используя сформированный массив, вывести на экран дисплея информацию о поездах, отправляющихся в Ленинград от 21 до 24 часов.

Вариант 2.

a)       Сформировать массив «Ассортимент», содержащий сведения об игрушках: указывается название игрушки, ее стоимость и возрастные границы.

b)       Вывести на экран содержимое массива в виде таблицы с заголовками столбцов.

c)        Получить следующие сведения: названия игрушек, которые подходят детям 5 лет и цена за которые не превышает $4.

Вариант 3

a)       Сформировать массив, содержащий различные даты. Каждая дата - это число, месяц и год.

b)       Вывести на экран содержимое массива в виде таблицы с заголовками столбцов.

c)        Найти: а) год с наименьшим номером; б) все весенние даты; в) самую позднюю дату.

Вариант 4

a)       Заполнить массив «Товар», содержащий сведения об экспортируемых товарах: указывается наименование товара, страна, импортирующая товар, и объем поставляемой партии в штуках.

b)       Вывести на экран содержимое массива в виде таблицы с заголовками столбцов.

c)        Получить список стран, в которые экспортируется товар, введенный с клавиатуры, и общий объем его экспорта.

Вариант 5

a)       Сформировать массив, содержащий сведения о книгах (фамилия автора, название и год издания).

b)       Вывести на экран содержимое массива в виде таблицы с заголовками столбцов.

c)        Найти название книг данного автора, изданных с 1990 г.

d)       Определить, имеется ли книга с названием «ИНФОРМАТИКА» Если да, то сообщить фамилию автора и год издания. Если таких книг несколько, то сообщить имеющиеся сведения обо всех этих книгах.

Вариант 6

a)  Сформировать массив сведений об учениках (фамилия, имя, номер класса).

b)  Вывести на экран содержимое массива в виде таблицы с заголовками столбцов.

c)   Осуществить перевод учеников на один класс вперед, т.е. первоклассников перевести во второй, второклассников в третий и т.д.

Файлы записей

По условию задачи разработать набор программ (или подпрограмм), позволяющих выполнить следующие действия:

1.  Создать в текущем каталоге типизированный файл для заданной структуры записи и заполнить его записями с клавиатуры по заданию.

2.  Вывести содержимое файла на экран в виде таблицы с заголовками столбцов.

3.  Добавить одну дополнительную запись с клавиатуры в конец ранее созданного файла.

4.  Осуществить поиск (выборку) записей из файла, удовлетворяющих заданному критерию (критерии для поиска можно определять самостоятельно).

5.  Удалить запись с указанным номером из файла.

6.  Выполнить сортировку записей в файле по наиболее подходящему полю.

После отладки и проверки работоспособности разработанных программ написать одну программу, которая позволяет организовать работу с базой данных указанной в условии структуры посредством меню. Меню должно содержать пункты, перечисленные выше.

Вариант 1

Создать файл, содержащий сведения о пациентах поликлиники. Структура записи: ФИО пациента, год рождения, адрес, диагноз. Выбрать информацию о пациентах, обратившихся с диагнозом "грипп".

Вариант 2

Создать файл, содержащий сведения о книгах,  находящихся в читальном зале библиотеки. Каждая запись должна содержать: фамилию и инициалы автора, название книги, место издания, наименование издательства, год издания, количество страниц. Выбрать записи о книгах издательства "Высшая  школа", выпущенных после 1990 года.

Вариант 3

Создать файл, содержащий информацию о свободных местах в поездах. Запись содержит: дату выезда, номер рейса, конечный пункт назначения, время отправления, число свободных купейных мест, число свободных плацкартных мест. Выдать информацию об имеющихся свободных местах на определенное число.

Вариант 4

Создать файл, содержащий сведения об учениках. Каждая запись должна содержать: номер по списку, фамилию, имя, отчество; год рождения; адрес. Выбрать сведения об учениках, проживающих на улице Книповича.

Вариант 5

Создать файл, содержащий сведения о предметах, изучаемых в 10 классе. Запись должна содержать: название предмета, общее количество часов на год, количество часов в неделю. Выбрать информацию о предметах, количество часов в неделю для которых больше 4.

Вариант 6

Создать файл, содержащий сведения о компьютерах. Структура записи: тип ПК, тактовая частота,  об"ем ОП, об"ем винчестера, стоимость. Выбрать  информацию о компьютерах, имеющих об"ем ОП более  2 Мбайт.

Вариант 7

Создать файл, содержащий сведения об итогах сдачи экзамена по информатике. Каждая запись содержит  фамилию и имя учащегося, годовую оценку, оценку за экзамен, итоговую оценку. Выбрать информацию об учащихся, повысивших годовую оценку на экзамене.

Вариант 8

Создать файл, содержащий сведения о результатах чемпионата по футболу. Структура записи: названия  команд, результаты игры. Выбрать информацию  о результатах игр команды "Спартак".

Вариант 9

Сформировать файл, содержащий сведения о магазинах округа. Структура записи: название магазина, его номер, адрес, фамилия директора. Выбрать информацию о магазинах, расположенных на улице П.Зори.

Вариант 10

Сформировать файл, содержащий сведения о владельцах автомобилей: фамилия владельца, марка автомобиля, номер. Выбрать информацию о владельцах автомобилей, имеющих заданную марку автомобиля.

Дополнительное задание

1.    Составить программу, которая  создает в текущем каталоге файл Books.dat и заполняет его записями о книгах магазина.

2.       Составить программу, которая считывает данные из файла books.dat и:

a.         выводит их на экран в виде таблицы.

b.       Выводит на экран только сведения о тех книгах, количество которых меньше 100.

c.        Вычисляет среднюю стоимость книг и выводит ее на экран.

3.       Написать программу, которая считывает сведения из исходного файла книг, подсчитывает количество книг по разделам и записывает результат в файл Itog.txt в текущем каталоге в виде:

Проза: 1

Поэзия: 2

Детектив: 1

Учебник: 3

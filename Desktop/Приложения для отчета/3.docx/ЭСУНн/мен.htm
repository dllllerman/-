Эффективным способом хранения и обработки большого количества однотипных переменных в программе являются массивы. Но у них есть недостаток: они не пригодны для долговременного хранения информации. По окончании работы программы все данные будут потеряны. Файл – это набор данных, хранящихся во внешней памяти ПК под заданным именем.

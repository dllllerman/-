Задачи для самостоятельного решения

Вариант 1

1.       Заполнить с клавиатуры одномерный массив из 5 целых чисел и вывести количество ненулевых элементов. Перед вводом каждого элемента на экране должна появляться подсказка с его номером.

2.       Заполнить массив из 10 элементов случайными числами. Вычислить среднее арифметическое всех его положительных элементов.

3.       Заполнить массив из 10 элементов случайными числами. Увеличить каждый его элемент в 2 раза.

4.       Заполнить линейный массив из 10 элементов случайными числами. Найти минимальный элемент массива и вывести его на экран с пояснениями.

5.       Написать программу, которая вычисляет, сколько раз введенное с клавиатуры число встречается в массиве из 10 случайных чисел.

6.       Заполнить массив случайными числами. Вывести на экран его положительные элементы красным цветом, а отрицательные – синим.

7.       Заполнить двумерный массив числами с клавиатуры и вывести его на экран в виде таблицы.

8.       Заполнить двумерный массив случайными числами и вывести его на экран в виде таблицы.

9.       В массиве из предыдущей задачи отыскать максимальный элемент, вывести его на экран с комментарием и указать, в какой строке и каком столбце он расположен.

10.  В массиве из предыдущей задачи найти сумму элементов по столбцам.

11.  Заполнить квадратную матрицу случайными числами. Найти сумму элементов главной диагонали.

12.  Заполнить таблицу целыми числами. Отыскать в ней максимальный и минимальный элементы. Поменять их местами. Вывести полученную таблицу на экран.

13.  Квадратный массив NxN заполнить следующим образом: элементы главной диагонали (слева направо сверху вниз) заполнены единицами. Ниже главной диагонали – нули, а элементы выше равны сумме своих индексов.

14.  В заданной целочисленной матрице A(N, M) найти количество строк, содержащих нули.

15.  В заданной матрице A(N, M) поменять местами строки с номерами P и Q
(1 <= P <= N, 1 <= Q <= N).

Вариант 2

1.       Заполнить с клавиатуры одномерный массив из 5 целых чисел и вывести количество нулевых элементов. Перед вводом каждого элемента на экране должна появляться подсказка с его номером.

2.       Заполнить массив из 10 элементов случайными числами. Вычислить среднее арифметическое всех его ненулевых элементов.

3.       Заполнить два массива (из 10 элементов каждый) случайными числами. Сформировать третий массив, каждый элемент которого равен полусумме соответствующих элементов исходных массивов.

4.       Заполнить линейный массив из 10 элементов случайными числами. Найти максимальный элемент массива и вывести его на экран с пояснениями.

5.       Написать программу, которая вычисляет, сколько раз введенное с клавиатуры число встречается в массиве из 10 случайных чисел.

6.       Заполнить линейный массив случайными числами. Вывести на экран его положительные элементы красным цветом, а отрицательные – синим.

7.       Заполнить двумерный массив числами с клавиатуры и вывести его на экран в виде таблицы.

8.       Заполнить двумерный массив случайными числами и вывести его на экран в виде таблицы.

9.       В массиве из предыдущей задачи отыскать минимальный элемент, вывести его на экран с комментарием и указать, в какой строке и каком столбце он расположен.

10.  В массиве из предыдущей задачи найти сумму элементов по строкам.

11.  Заполнить квадратную матрицу случайными числами. Найти среднее арифметическое элементов главной диагонали.

12.  Заполнить таблицу целыми числами. Отыскать в ней максимальный и минимальный элементы. Поменять их местами. Вывести полученную таблицу на экран.

13.  По трём заданным матрицам А(N, N), В(N, N) и С(N, N) постройте матрицу Х того же размера, каждый элемент которой представляет собой среднее арифметическое соответствующих элементов исходных матриц.

14.  Дана матрица A(N, N). Если хотя бы один элемент строки матрицы отрицателен, то все элементы этой строки заменить нулями.

15.  Заполнить квадратный массив случайными числами. Организовать два вектора. В один записать элементы матрицы, лежащие выше главной диагонали, в другой – ниже главной диагонали.

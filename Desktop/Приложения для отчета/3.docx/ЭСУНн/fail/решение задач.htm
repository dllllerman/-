Условие задачи 1. Создадим текстовый файл с именем file01.txt, содержащий одну строку текста, которая записывается в него в виде текстовой константы.

Список переменных: f – файловая переменная, st – строка, прочитанная из текстового файла.

program text_file_1;

var f:text; st:string;

begin

assign(f,'file01.txt');

rewrite(f);

close(f);

assign(f,'file01.txt');

reset(f);

readln(f,st);

close(f);

writeln('Проверка ввода/вывода в файл:');

end.

Условие задачи 2. Составить программу, которая заполняет текстовый файл file.txt набором строк с клавиатуры, выводит его содержимое на экран, подсчитывает количество символов и строк в данном текстовом файле.

Список переменных: f – файловая переменная, stroka – буферная переменная для временного хранения строк, sumstr – количество строк, sumchr – количество символов.

Листинг 2

program text_file_2;

const sumstr:word=0;

sumchr:word=0;

var f:text;

stroka:string;

begin

writeln('Введите текст из строк, ввод закончите символом #');

assign(f,'file.txt');

{Создание текстового файла, запись в него набора строк с клавиатуры}

rewrite(f);

repeat

readln(stroka);

if (stroka<>'#') then writeln(f,stroka); {запись строки в файл}

until stroka='#';

close(f);

{открытие, вывод и обработка файла}

reset(f);

writeln('Файл содержит:');

while not eof(f) do

begin

readln(f,stroka);   {чтение строки из файла}

writeln(stroka);

inc(sumstr);                   {подсчет числа строк}

inc(sumchr,length(stroka)); {подсчет числа символов}

end;

writeln('В файле: строк - ', sumstr,' символов - ', sumchr);

close(f);

end.

Условие задачи 4. Добавление строк с клавиатуры в начало существующего текстового файла.

Список переменных: f – файловая переменная (исходный текстовый файл), ft – файловая переменная (временный текстовый файл), name – имя исходного файла, buf – буферная переменная для временного хранения строк.

Листинг 4

program Title;

var ft,f:text;

buf:string;

name:string;

begin

{Запись текста с клавиатуры во временный файл на диск}

assign(ft,'tmp.txt'); rewrite(ft);

writeln('Вводите текст для записи.Окончание - Enter');

repeat

write('->'); read(buf); writeln(ft,buf);

until buf='';

close(ft); writeln('Ввод окончен');

writeln('Введите имя файла для добавления');

read(name);

{Добавление строк из исходного файла в конец временного файла}

assign(ft,'tmp.txt'); append(ft);

assign(f,name+’.txt’); reset(f);

while not eof(f) do

begin

readln(f,buf);

if buf <> '' then writeln(ft,buf);

end;

close(f); close(ft);

{Удаляем исходный файл и переименовываем временный}

erase(f); rename(ft,name+’.txt’);

end.

Пояснение к коду: создается новый временный файл tmp.txt, в него записываются данные, которые необходимо разместить в начале существующего файла name. Файл с именем name открывается для чтения. Для дозаписи в конец при помощи append открывается файл tmp.txt. В цикле происходит чтение файла name и дозаписывание файла tmp.txt строка за строкой. Затем происходит удаление файла name, а tmp.txt переименовывается в name.

Условие задачи 8. Подсчитать количество положительных и отрицательных элементов файла целых чисел, результаты подсчета вывести на экран и записать в текстовый файл Result.txt.

Список переменных: f – файловая переменная (исходный файл с целыми числами), r – файловая переменная (текстовый файл с результатом обработки), buf – буферная переменная для временного хранения чисел, name – имя файла, np и nn – количество положительных и отрицательных элементов исходного файла.

Листинг 8

program tip_file_4;

uses crt;

var    f:file of integer;

r:text;

name:string;

buf:integer;

np,nn:byte;

begin

write('Имя файла:'); read(name);

//подготовка файлов к работе

assign(f,name+'.dat'); reset(f);

assign(r,'Result.txt'); rewrite(r);

//подсчет элементов

np:=0; nn:=0;

while not eof(f) do

begin

read(f,buf);

if buf>0 then np:=np+1 else nn:=nn+1;

end;

close(f);

//вывод результатов на экран

writeln('Положительных чисел в файле ',name,'.dat:',np);

writeln('отрицательных чисел в файле ',name,'.dat:',nn);

//вывод результатов в текстовый файл

writeln(r,'Положительных чисел в файле ',name,'.dat:',np);

writeln(r,'отрицательных чисел в файле ',name,'.dat:',nn);

close(r)

end.

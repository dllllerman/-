10.         Задачи для самостоятельного решения

Текстовые файлы

Вариант 1

1.       Создать текстовый файл, имя которого запрашивается с клавиатуры. Заполнить его набором строк, вводимых с клавиатуры. Отыскать в файле самую длинную строку. Вывести ее на экран и одновременно записать в текстовый файл result.txt.

2.       Создать текстовый файл, имя которого запрашивается с клавиатуры. Заполнить его набором строк, вводимых с клавиатуры. Сформировать файл result.txt, содержащий строки из исходного файла, в начало каждой строки должен быть записан ее порядковый номер, а в конец строки число, показывающее количество символов в строке.

Вариант 2

1.       Создать текстовый файл, имя которого запрашивается с клавиатуры. Заполнить его набором строк, вводимых с клавиатуры. Отыскать в файле все строки заданной длины. Вывести их на экран и одновременно записать в текстовый файл result.txt.

2.       Создать текстовый файл, имя которого запрашивается с клавиатуры. Заполнить его набором строк, вводимых с клавиатуры. Записать в перевернутом виде строки из исходного файла в файл result.txt.

Типизированные файлы

Задача 1.                   

a.        Ввести с клавиатуры целое число N (>1). Создать файл целых чисел и записать в него N первых положительных четных чисел (2, 4, …).

b.       Вывести содержимое файла на экран.

Задача 2.                   

a.        Создать файл и записать в него произвольное количество целых чисел (окончание ввода 0).

b.       Вывести на экран все содержимое файла, а затем первый, второй, предпоследний и последний элементы данного файла.

Задача 3.                   

a.        Заполнить исходный файл вещественными числами (окончание ввода 0).

b.       Вывести на экран все содержимое файла. Переписать из него в другой файл первый и последний элементы. Вывести на экран содержимое полученного файла.

Задача 4.                   

a.        Создать файл и заполнить его целыми  числами (окончание ввода 0).

b.       В другой файл переписать все его элементы в обратном порядке. Вывести на экран содержимое исходного и полученного файлов с комментариями.

Задача 5.                   

a.        Создать файл и заполнить его целыми числами (окончание ввода 0).

b.       Образовать два новых файла, первый из которых содержит элементы исходного файла с четными номерами, а второй – с нечетными. Вывести на экран содержимое всех трех файлов с комментариями.

Задача 6.                   

a.        Создать файл и заполнить его целыми числами (окончание ввода 0).

b.       Вывести на экран все содержимое файла. Найти среднее арифметическое его элементов.

Задача 7.                   

a.        Создать файл и заполнить его целыми числами (окончание ввода 0).

b.       Вывести на экран все содержимое файла. Найти сумму его элементов с четными номерами.

Задача 8.                   

a.        Создать файл и заполнить его целыми числами (окончание ввода 0).

b.       Создать второй файл, в который записать квадраты элементов первого файла. Вывести на экран содержимое файлов с комментариями.

Задача 9.                   

a.        Создать файл и заполнить его целыми числами (окончание ввода 0).

b.       Вывести на экран все содержимое файла. Создать второй файл, в который записать значения квадратных корней из элементов первого файла. Вывести на экран все элементы полученного файла.

Задача 10.              

a.        Создать файл и заполнить его целыми числами (окончание ввода 0).

b.       Удалить из него все отрицательные элементы. Вывести на экран исходный и конечный вид файла с комментариями.

Задача 11.              

a.        Создать файл и заполнить его целыми числами (окончание ввода 0).

b.       Удвоить размер файла, записав в конец файла все его исходные элементы в том же порядке. Вывести на экран исходный вид файла и конечный с комментариями.

Задача 12.              

a.        Создать файл и заполнить его целыми числами (окончание ввода 0).

b.       Заменить в нем каждое положительное число на три нуля. Вывести на экран исходный и конечный вид файла с комментариями.

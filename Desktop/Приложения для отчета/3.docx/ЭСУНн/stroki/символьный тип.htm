1.    Символьный тип

Значением переменных символьного типа является один символ. Каждому символу соответствует код символа – целое число в диапазоне от 0 до 255. Символьный тип является порядковым. Все действия по обработке символов сводятся к действиям над целыми числами, расположенными строго по порядку.

Над данными символьного типа определены операции отношения: =, <>, <, >, <=, >=, вырабатывающие результат логического типа. При выполнении операций сравнения коды символов сравниваются как обычные целые числа.

2.    Строковый тип

Строка – последовательность символов. Максимальное количество символов в строке (длина строки) - 255. Переменную строкового типа можно определить в разделе определения типов:

Type           ИмяТипа = string [макс.длина строки];

Var   Идентификатор : ИмяТипа;

или в разделе объявления переменных:

Var  Идентификатор : string [макс.длина строки];

Var  Идентификатор : string;

Если максимальная длина строки не указывается, то она равна 255 байт.

Строковые данные могут использоваться в программе и в качестве констант: Const Str_Adres = ‘ул. Мира, 35’;

Строка в языке Паскаль трактуется как массив символов. Для строки из n символов в памяти отводится n+1 байт; n байтов – для хранения символов строки, а один – для значения текущей длины строки. Этот дополнительный байт имеет номер 0, соответственно первый символ строки имеет номер 1, второй – 2 и т.д.

К любому символу в строке можно обратиться как к элементу одномерного массива array [0..n] of char по номеру (индексу) данного символа в строке. В отличие от массивов переменные строкового типа могут участвовать целиком в операторах ввода-вывода.

Примеры решения задач


Условие задачи 1. Заполнить линейный массив вещественными числами с клавиатуры. В векторе найти номера и значения первого минимального и последнего максимального элементов.

Рис. 1. Алгоритм поиска максимального и минимального элементов одномерного массива

Список переменных: a – линейный числовой массив, n – число элементов массива, i – индекс элемента массива, nmin – номер минимального элемента массива, nmax – индекс максимального элемента массива, min – значение минимального элемента массива, max – значение максимального элемента массива.

В алгоритме решения задачи (рис. 4.1) можно выделить два крупных блока, в каждом из которых необходимо применение цикла с параметром для просмотра всех элементов массива: заполнение массива и вывод его на экран, поиск максимального и минимального элементов.

Листинг 4.1

program min_and_max;

uses CRT;

const n=10;

var    a: array [1 .. n] of real;

i, nmin, nmax: byte;

min, max: real;

begin

writeln ('Введите элементы одномерного массива:');

for i:=1 to n do

begin

write ('a[',i,']=');

read (a[i]);

end;

min:=a[1]; nmin:=1;

max:=a[1]; nmax:=1;

for i:=1 to n do

begin

if a[i] < min then

begin

min:=a[i]; nmin:=i

end;

if a[i] >= max then

begin

max:=a[i]; nmax:=i

end

end;

writeln ('min=', min:4:2,',его индекс=', nmin:3);

writeln ('max=', max:4:2,',его индекс=', nmax:3);

end.

Условие задачи 2. Сформировать линейный массив из N случайных целых чисел, распределенных в диапазоне от -50 до 50. Вычислить среднее арифметическое положительных элементов массива.

Список переменных: A – линейный числовой массив, Nmax – максимально возможное число элементов массива (для резервирования памяти), N – число элементов массива, i – индекс элемента массива, Sp – сумма положительных элементов массива, Np – количество положительных элементов массива, sr – среднее арифметическое положительных элементов массива.

На рисунке 4.2 представлена блок-схема алгоритма решения задачи.

Рис. 2. Алгоритм вычисления среднего арифметического положительных элементов одномерного массива

Листинг 2

program Sred_Pol;

uses crt;

const nmax=100;

type x = array [1 .. nmax] of integer;

var    a: x;

i, n, np: byte;

Sp, sr:real;

begin

write ('Число элементов формируемого массива->'); read (n);

{Формирование массива}

for i:=1  to n do

begin

a[i]:=random (100)-50;

write (a[i]:4)

end;

writeln;

{Вычисление суммы и количества положительных элементов массива}

sp:=0; np:=0;

for i:=1 to n do

if a[i]>0 then

begin

sp:=sp+a[i]; np:=np+1;

end;

sr:=sp/np;

writeln ('Ср.арифметическое положительных эл-в массива = ',sr:4:2)

end.

Условие задачи 3. Сформировать числовую матрицу, элементами которой являются произведения соответствующих номеров строк i и номеров столбцов j.

Список переменных: P – числовая матрица, ni – число строк матрицы, nj – число столбцов матрицы, i – номер строки матрицы, j – номер столбца матрицы.

В алгоритме заполнения матрицы используются вложенные циклы, с помощью которых организован просмотр всех строк и столбцов матрицы (рис.3).

Рис. 3. Алгоритм формирования матрицы из произведений соответствующих номеров строк и столбцов

Листинг 3

Условие задачи 4. Из прямоугольной матрицы A[n,m], заполненной случайными числами, сформировать вектор В, элементами которого являются наибольшие элементы столбцов матрицы.

Алгоритм решения задачи представлен на рисунке 4.

Рис. 4. Алгоритм формирования вектора из максимальных элементов толбцов матрицы

Список переменных: a – числовая матрица, n – число строк матрицы, m – число столбцов матрицы, b – числовой вектор, i – номер строки матрицы, j – номер столбца матрицы (индекс элементов вектора b).

Листинг 4

program Matr_Vector;

uses Crt;

const n=5; m=7;

var    a:array[1..n,1..m] of integer;

b:array[1..m] of integer;

i, j: integer;

begin

{заполнение матрицы A случайными числами}

writeln ('Сформированная матрица:');

for i:=1 to n do

begin

for j:=1 to m do

begin

a[i, j]:=Random(100)-50;

write (a[i, j]:4)

end;

writeln

end;

{формирование вектора B из максимальных элементов столбцов матрицы}

for j:=1 to m do

begin

b[j]:=a[1, j];

for i:=2 to n do

if b[j]<a[i, j] then b[j]:=a[i, j];

end;

{вывод вектора B на экран}

writeln ('Числовой вектор В:');

for i:=1 to m do write(b[i]:4);

end.

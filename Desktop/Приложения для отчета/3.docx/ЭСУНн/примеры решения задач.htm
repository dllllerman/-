5.    Примеры решения задач

Условие задачи 1. Написать программу, которая запрашивает слово (строку символов) с клавиатуры и выводит на экран его перевертыш.

Список переменных: s – слово (строка символов), введенное с клавиатуры, sp – слово-перевертыш, len – длина слова S (количество символов в слове), i – порядковый номер символа в строке S.

Для образования строки-перевертыша необходимо перебирать исходную строку по буквам, начиная с последней, и каждую букву присоединять к результирующей строке. Таким образом, для работы со строкой нужно применить циклическую конструкцию (рис. 5.1), удобно использовать цикл с параметром, так как длина исходной строки (количество букв) заранее известна.

Рисунок 1 - Алгоритм построения перевертыша строки

Листинг 1

program perevert;

uses crt;

var s, sp : string; len, i : byte;

begin

write ('Введите слово>');

read (S);

len:=length(s); //определение длины слова

{формирование перевертыша}

sp:=’’;

for i:=len downto 1 do

begin

sp:=sp+s[i];

end;

write ('Слово-перевертыш:', sp);

    end.

Рисунок 2 -Алгоритм обнаружения слов в массиве, начинающихся и заканчивающихся гласной буквой

Условие задачи 2. В массиве из слов (символьных строк), введенных с клавиатуры, подсчитать количество слов, начинающихся и заканчивающихся гласной буквой.

Список переменных: maxn – максимальное количество слов в массиве, gl – строка из гласных букв, s – линейный массив из слов, n – количество слов массива S, i – номер слова в массиве S, len – длина отдельного слова, k – количество слов, которые начинаются и заканчиваются гласной буквой.

Листинг 2

program glasn;

uses crt;

const maxn = 100; gl = ’аоиуеыэюяАОИУЕЫЭЮЯ’;

var    s: array [1..maxn] of string;

len, n, i, k : integer;

begin

{Заполнение массива словами с клавиатуры}

write ('Введите количество слов массива->');

read (n);

writeln ('Введите элементы массива s (набор слов)');

for i:=1 to n do read (s[i]);

{Обработка массива, поиск слов в соответствии с заданием}

k:=0;

writeln(‘Слова начинающиеся и заканчивающиеся гласной буквой:’);

for i:=1 to n do

begin

len:= length (s[i]);

if (pos(s[i][1],gl) > 0)and (pos(s[i][len],gl) > 0)then

begin

k:=k+1;

writeln(s[i])  

end

end;

writeln ('Всего слов -> ', k);

end.

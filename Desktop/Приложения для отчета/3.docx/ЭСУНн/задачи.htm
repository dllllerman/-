6.    Задачи для самостоятельного решения

Вариант1

1.       Программа запрашивает слово и выводит на экран коды каждой буквы данного слова.

2.       Определите долю пробелов в заданной строке.

3.       Введённый текст напечатать, выделив цветом все гласные.

4.       Каждые n символов во введенном тексте отделите знаком "!".

5.       Если брать часть слова без первой буквы, без двух первых букв и так далее, получится: «Победа» – «обеда» – «беда» – «еда» – «да» – «а». Составить программу, которая проделывает это с введённым словом или предложением.

6.       Вычеркните из заданного слова все буквы "а".

7.       Написать программу, которая запрашивает предложение и формирует массив из слов данного предложения.

8.       Дан массив из слов A(n). Найти все слова, начинающиеся  с заданной буквы.

Вариант 2

1.       Вывести на экран первую часть таблицы кодировки, то есть символы, коды которых начинаются с нуля и заканчиваются 127.

2.       В заданном тексте подсчитайте общее количество букв "x" и "y".

3.       Заменить в слове все буквы "а" на "б".

4.       Написать программу, которая в русской фразе, введенной с клавиатуры, преобразует строчные буквы в прописные.

5.       Если брать часть слова без первой буквы, без двух первых букв и так далее, получится: «Победа» – «обеда» – «беда» – «еда» – «да» – «а». Составить программу, которая проделывает это с введённым словом или предложением.

6.       Вычеркните из заданного слова все буквы "а".

7.       Написать программу, которая запрашивает предложение и формирует массив из слов данного предложения.

8.       Дан массив из слов Т(к). Найти все слова, содержащие заданное количество букв.
